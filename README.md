# sparkbiscuit.me

Source for [sparkbiscuit.me](https://sparkbiscuit.me) — Filuma's home on the
web. A static GitHub Pages site: a small landing/support page and the
[privacy policy](https://sparkbiscuit.me/privacy/), styled to match the app's
Hearthlight design language.

The repo name must stay `Sparkbiscuit.github.io` — that's how GitHub Pages
serves a user site at the apex domain (via `CNAME`).
